## ğŸ’¡ Customization Tips

### Change Images
1. Find image on Unsplash
2. Get URL: `https://images.unsplash.com/photo-...?w=SIZE&q=80`
3. Replace URL in component
4. No rebuild needed for image changes

### Local Images
1. Add to `public/images/`
2. Reference as `/images/filename.jpg`
3. Better for production (faster, no external dependency)

### Image Sizes
- Hero: 1920x1080px recommended
- Properties: 800x600px recommended
- Keep aspect ratio for best results
